🛠️ Especificaciones Técnicas del ADN Emiratí
El contrato integra mejoras de rendimiento y lógica de rangos:

Optimización por Custom Errors: Uso de error RangoExcedido, error YaVotado y error IDInvalido para reducir drásticamente el consumo de gas en comparación con los strings tradicionales de require.

Nivel de Homogeneización (uint256): Mide el grado de fusión mecánica entre proteína y fibra. El Al Harees (98) es el valor de referencia para una matriz casi líquida.

Tiempo de Cocción Lenta (uint256): Validado para no exceder las 48 horas, reflejando procesos de desnaturalización proteica extrema.

Gobernanza Blindada: Persistencia del sistema de un solo voto por wallet para asegurar la integridad de los datos de la comunidad.
